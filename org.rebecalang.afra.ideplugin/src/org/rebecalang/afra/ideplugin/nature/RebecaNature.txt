REBECANATURE.JAVA DOCUMENTATION
================================

PURPOSE:
Eclipse project nature that identifies projects as Rebeca projects and manages project-specific configuration.

ECLIPSE VS AFRA:
- Implements Eclipse IProjectNature interface
- Afra-specific nature for Rebeca language projects

KEY METHODS:
- configure(): Called when nature is added to project
- deconfigure(): Called when nature is removed, cleans up builders
- getProject()/setProject(): Project association methods

IMPORTANT NOTES:
- Identifies projects as Rebeca projects in Eclipse
- Enables Rebeca-specific functionality for projects
- Works with SampleBuilder for project building
- Required for project-level Rebeca language support
- Referenced in plugin.xml for project nature configuration
